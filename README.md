# Tursina Yisehak — Personal Portfolio

A clean, modern, glassmorphic portfolio website built with pure HTML and CSS. No frameworks, no dependencies — just well-structured markup and styling.

🔗 **Repo:** [github.com/tuyy-99/portfolio1](https://github.com/tuyy-99/portfolio1)

## Preview

Open `index.html` in your browser to view locally.

## Features

- Glassmorphism design with gradient accents
- Animated background orbs
- Fully responsive (mobile, tablet, desktop)
- Skills section with interactive hover effects
- Direct links to GitHub, LinkedIn, and email
- Real profile picture

## Project Structure

```
portfolio1/
├── index.html     # Main HTML structure
├── style.css      # All styles including responsive
├── pp.jpg         # Profile picture
└── README.md      # You're reading it
```

## Tech Stack

- HTML5
- CSS3 (custom properties, backdrop-filter, flexbox, grid)
- Font Awesome 6 (icons)
- Google Fonts — Inter

## Getting Started

No build step needed.

```bash
git clone https://github.com/tuyy-99/portfolio1.git
cd portfolio1
# open index.html in your browser
```

## Customization

| What | Where |
|------|-------|
| Name, bio, links | `index.html` |
| Colors, fonts, spacing | `style.css` (`:root` variables) |
| Profile picture | Replace `pp.jpg` |
